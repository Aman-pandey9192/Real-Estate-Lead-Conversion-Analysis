# 🏠 Real Estate Lead Conversion Analysis

## 📌 Project Overview

This project analyzes over **700,000 real estate lead records** to evaluate sales funnel performance, identify conversion bottlenecks, measure agent and channel partner effectiveness, and generate actionable business insights.

The analysis was conducted using **Python (Pandas, NumPy, Matplotlib)** for Exploratory Data Analysis (EDA) and **Power BI (DAX)** for dashboard development and KPI tracking.

---

## 🎯 Business Problem

Real estate organizations generate thousands of leads every month, but only a small percentage convert into bookings.

This project aims to answer key business questions:

- How effectively are leads moving through the sales funnel?
- Which agents generate the most bookings?
- Which channel partners contribute the most site visits?
- Which projects and locations perform best?
- Where are the biggest conversion losses occurring?

## 📈 Key Findings

### Lead Funnel Performance

- Majority of leads were not successfully connected.
- Only **4.7%** of total leads showed genuine interest.
- Site visit conversion remained below **1%**.
- Final booking conversion rate was approximately **0.09%**.

### Agent & CP Performance

- Top-performing agents significantly outperformed their peers.
- A small group of channel partners generated the majority of site visits.

### Customer Engagement

- Average call duration increased as leads progressed through the funnel.
- Booked customers had the longest average interaction time.
